# uitstapje_poll
Dit is een tijdelijke repository waar studenten kunnen stemmen op welke uitstapje ze willen maken voor les week 7 of 8.

Iedere student krijgt 1 stem. Stemmen doe je aan de hand van een txt bestand met je naam als titel. bijvoorbeeld: John.txt 
Zet je bestand in de branch die je interreseert.
De branch met de meeste stemmen wordt de 7de of 8ste les. Zijn er geen stemmen, kies ik het onderwerp.

Keuzes zijn: 
* Git als portofolio en CV: Github paginas worden vaak als portofolio gebruikt maar kunnen ook ingesteld worden als een CV. Dit zorgt voor dat als je soliciteert je de github kan delen en je werkgever bewijs kan laten zien van je werk en ervaring
* Git CLI: We gaan kijken naar de oorsprong van GIT waarbij we gaan werken met de Command line, zeer nuttig als je met versiebeheer op apparaten moet werken waar je niet een graphische interface op hebt, denk aan chips in apparatuur.
* GitHub actions: Actions laat je toe om bestanden gelijk te activeren en in gebruik kan nemen. Hierbij gaan we bestanden gelijk connecten aan een ftp server.

Let op:
* 2 of meer stemmen betekent dat al je stem wordt verwijderd en dat je niet meer mag stemmen
* de inhoud en titel van de commits moeten netjes zijn, ongepast taalgebruik betekend dat je stem verwijderd wordt en dat je niet meer mag stemmen.
* de inhoud van je txt bestand moet netjes zijn en geen ongepast taalgebruik bevatten. je stem wordt verwijderd en je mag niet meer stemmen
* Je zit niet aan een anders stemmen, doe je dit wel dan wordt jouw stem verwijderd en mag je niet meer stemmen.

Je kan stemmen op een onderwerp tot en met 3 december.
